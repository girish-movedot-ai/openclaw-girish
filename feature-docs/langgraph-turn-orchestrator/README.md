# LangGraph Turn Orchestrator

This folder contains all internal and experimental documents for the LangGraph turn-orchestrator feature.

## Structure

- `spec/` contains the source documents provided by the human operator
- `deliverables/` contains the repo-discovery and implementation-planning documents generated by the first agent

## Intended workflow

1. Read everything in `spec/`
2. Generate the requested documents into `deliverables/`
3. Review and commit the deliverables
4. Use a separate agent to implement the feature from the reviewed deliverables
